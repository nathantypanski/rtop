# rtop ![travis-ci-status](https://travis-ci.org/nathantypanski/rtop.svg?branch=master)

![screenshot](images/graph-2014-july.png)

A system monitor written in Rust.

To build and run:

    make
    make run

Or, if you have cargo,

    cargo build

## Features working

- Color graphs of one CPU

## Planned features

- Multiple CPUs
- RAM graphs
- Process usage
